# Landing Page Dashboard

דשבורד אינטראקטיבי להצגת נתוני אנליזה של עמוד נחיתה, מבוסס React, Chakra UI ו-Recharts.

## הרצה מקומית

1. התקן את התלויות:
   ```bash
   npm install
   ```
2. הפעל את הסביבה:
   ```bash
   npm start
   ```

## תיאור
הדשבורד מציג KPI, גרפים, טבלאות ותובנות מתוך קובץ ה-JSON `landing_page_analysis_json.json`.
